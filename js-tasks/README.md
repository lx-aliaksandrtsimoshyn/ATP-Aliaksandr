# JavaScript Tasks

This folder contains solutions to JavaScript coding challenges from the ATP platform.

## How to Run

1. Navigate to the js-tasks folder
2. Run individual files with: `node task-name.js`
3. Or run with TypeScript: `npx ts-node task-name.ts`

## Task Structure

Each task file contains:
- Problem description (commented at the top)
- Solution implementation
- Test cases
- Expected output

## Testing

You can add tests for JavaScript tasks using Playwright's test runner if needed.
